# Crypto Portfolio Optimization

This project analyzes cryptocurrency portfolio diversification using:

- Mean-variance optimization
- Monte Carlo simulations
- Risk metrics (Sharpe ratio, drawdown, volatility)

## Features

- Portfolio optimization
- Control vs optimized portfolio comparison
- Monte Carlo simulations
- Portfolio path simulations
- Risk visualization

## Technologies

- Python
- NumPy
- Pandas
- SciPy
- Matplotlib
